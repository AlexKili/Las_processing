# Las_processing
The free simple program to load and plot las-file based on lasio library

# Installation
Copy git to your computer and run Main.py

# Requirments packages:
    - PyQt5
    - PyQtWebEngine
    - lasio
    - plotly


# Usage
    Run Main.py
    Load las-file (menu tools or bar tools)
    Plot curves or save it
